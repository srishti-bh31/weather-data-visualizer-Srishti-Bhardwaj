# weather-data-visualizer-Srishti-Bhardwaj

This project analyses real-world weather data using Python.  
It includes data cleaning, statistics, grouping, and visualizations.

## Features
- Load and inspect real CSV weather data
- Clean missing values
- Convert date columns to datetime
- Compute daily & monthly statistics using NumPy
- Plot temperature, rainfall, humidity using Matplotlib
- Group data by months and seasons
- Export cleaned CSV
- Save all plots as PNG

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook / VS Code

## Dataset
Use any real open-source dataset from:
- Kaggle
- IMD
- Local weather records
